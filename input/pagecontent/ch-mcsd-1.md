This section corresponds to transaction [CH:mCSD-1]. Transaction [CH:mCSD-1] is used by the Provider Information 
Source and Provider Information Directory Actors.

### Scope

The Update Matching Care Services [CH:mCSD-1] transaction is used to update document metadata from the Provider 
Information Source to the Provider Information Directory.

### Actor Roles

**Actor:** Care Services Source   
**Role:** Prepares and issues an add, update, or delete request to Care Services resources
**Actor:** Care Services Directory    
**Role:** Accepts add, update, or delete requests to Care Services resources

### Referenced Standards

1. mCSD
2. HPD
3. This MHD Profile is based on Release 4 of the [HL7® FHIR®](https://hl7.org/fhir/R4/index.html) standard.
